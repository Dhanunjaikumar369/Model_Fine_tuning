# YouTube Video Search Titles Generator Using Hugging Face LLM

This project generates YouTube search titles related to Indian two-wheeler reviews, feedback, and launches. It leverages the Hugging Face API to interact with a large language model (LLM) and generate search suggestions for relevant video topics based on a given prompt.

## Features

* **LLM Integration** : Uses Hugging Face's InferenceClient with the LLaMA-3.1-70B model to generate search titles based on user-defined prompts.
* **Environment Variables** : Supports environment variables for secure API key management.
* **Modular Functions** : Includes reusable functions to interact with the LLM for generating responses.

## Prerequisites

To run this project, you'll need the following:

* Python 3.7+
* An API key from Hugging Face
* An API key from YouTube (if extending functionality)
